写b站小脚本的曲折

Thu, 24 Mar 2022

起因

如果你想做一些网页相关的小脚本，具体的思路大概有以下3种：

  • 1.定位到网页元素的具体位置，然后再操作
  • 2.定位到网页元素的css选择器，然后再操作
  • 3.找到接口，然后直接发请求

这时会遇到以下问题：

  • 1.元素位置可能会随着主站更新而变更
  • 2.有些数据是异步的，比如网络请求，事件等
  • 3.写接口需要密钥，需要拼接参数，伪造请求头

经过

上面的这些问题，可以通过以下的例子来体验并解决：

目标：批量删除b站动态

//找到所有的三点图标
let items = document.getElementsByClassName("more-button bp-icon-font icon-more-1"); 

//点击三点
items[0].click();

//点击置顶下的删除
document.getElementsByClassName("child-button c-pointer")[1].click();

//1ms后点击蓝色删除按钮
let finished = document.getElementsByClassName("bl-button panel-btn bl-button--primary bl-button--size")[0].click();

因为这些操作是异步的，可以把它们套在setTimeout(()=>{},0)

//找到所有的三点图标
let items = document.getElementsByClassName("more-button bp-icon-font icon-more-1"); 

//实现删除的函数
function delete_item(){
    //点击三点
    items[0].click();
    setTimeout(() => {
        //点击置顶下的删除
        document.getElementsByClassName("child-button c-pointer")[1].click();
    },0)
    setTimeout(() => {
        //1ms后点击蓝色删除按钮
        let finished = document.getElementsByClassName("bl-button panel-btn bl-button--primary bl-button--size")[0].click();
    },1)
    clearTimeout();
}

然后隔1s删一个,然后刷新页面


//功能：批量删除网页版b站动态
//使用：先把动态拉到底，将所有动态都加载出来，然后将下面代码复制粘贴到console，等待即可！

//找到所有的三点图标
let items = document.getElementsByClassName("more-button bp-icon-font icon-more-1"); 

//实现删除的函数
function delete_item(){
    //点击三点
    items[0].click();
    setTimeout(() => {
        //点击置顶下的删除
        document.getElementsByClassName("child-button c-pointer")[1].click();
    },0)
    setTimeout(() => {
        //1ms后点击蓝色删除按钮
        let finished = document.getElementsByClassName("bl-button panel-btn bl-button--primary bl-button--size")[0].click();
    },1)
    clearTimeout();
}
for(let i=0;i<items.length;i++){
    //每隔1秒删1个，避免b站限制
    setInterval(() => {delete_item()},1000);
    clearInterval();        
}
//15秒后自动刷新页面
setTimeout(() => {
    location.reload();
},15000)
