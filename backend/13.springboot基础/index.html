1.springboot将spring中各种繁琐的配置进行简化

以前写的ssm项目属于单体结构，springboot写的项目是微服务项目

  • @springbootApplication扫描spring.factories的全部自动配置

  • @condition并不会使它们全部生效，需要添加对应starter来启动

  • @xxxautoConfiguration/xxProperties扫描时会存在与yml配置文件绑定实现手动改配置

  • @Configuration在config/xxConfig.java自定义starter

2.springboot可以通过yml文件为实体类赋值

package wiki.mrx.pojo;


import lombok.AllArgsConstructor;
import lombok.Data;
import lombok.NoArgsConstructor;
import org.springframework.boot.context.properties.ConfigurationProperties;
import org.springframework.stereotype.Component;

@Component//把当前类注入spring容器中，相当于写了<bean>
@Data//lombook自动生成属性的get/set
@NoArgsConstructor//lombook自动生成无参构造函数
@AllArgsConstructor//lombook自动生成有参构造函数
@ConfigurationProperties(prefix = "person")//当前类与yml配置文件关联
@Validated//数据校验
public class Person {
	@Email(message = "邮箱格式错误")
    public String name;
    public int age;
}
