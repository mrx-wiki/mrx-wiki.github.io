java面向对象

Tue, 1 Jun 2021

先有逻辑正确的思路，剩余的是重复的实践与积累

1.面向对象

===========================================

1.1 类是有属性，方法和构造器的模版，对象是实例

===========================================

1.2 面向对象的三大特性：

===========================================

  • 封装：修饰符，get,set方法
  • 继承：extends，super，源头object
  • 多态：同一行为，不同子类有不同表现，继承，子类重写父类方法，父类引用指向子类实例
//父类
package wiki.mrx.www;

public class Person {

    //属性
    private String name;
    static int age;

    //构造器：用于初始化数据
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    //get方法
    public String getName() {
        return name;
    }

    public static int getAge() {
        return age;
    }
    //set方法
    public void setName(String name) {
        this.name = name;
    }

    public static void setAge(int age) {
        Person.age = age;
    }

    //自定义方法
    public void eat(){
        System.out.println("人可以吃饭");
    }
}

//Teacher子类
package wiki.mrx.www;

public class Teacher extends Person{
    //构造器
    public Teacher(String name, int age) {
        super(name, age);
    }
    //重写父类eat方法
    @Override
    public void eat() {
        System.out.println("老师在教师食堂吃饭");
    }
    //自定义方法
    public void teac(){
        System.out.println("老师能教学");
    }
}



//Student子类
package wiki.mrx.www;

public class Student extends Person{
    public Student(String name, int age) {
        super(name, age);
    }
    //重写父类eat方法
    @Override
    public void eat() {
        System.out.println("学生在学生食堂吃饭");
    }

    //自定义方法
    public void study(){
        System.out.println("学生可以学习");
    }
}


//测试类
package wiki.mrx.www;

public class Test {
    public static void main(String[] args) {
        Teacher t = new Teacher("王老师",33);
        Person p1 = t;
        Person p2 = new Student("小明",18);
        p1.eat();//老师在教师食堂吃饭
        p2.eat();//学生在学生食堂吃饭
        t.teac();

    }
}

===========================================

1.3 接口interface中的抽象方法是规范

实现类implements接口中重写的抽象方法是具体实现

接口相比于抽象类可以实现多继承

===========================================

//Student接口多继承person和cat接口
package wiki.mrx.www;

public interface Person {
    void eat();
}

interface Cat {
    void sleep();
}

interface Student extends Person, Cat{
    void study();
}

//StudentImp类是student接口的实现类，重写了他的抽象方法
package wiki.mrx.www;

public class StudentImp implements Student{
    public StudentImp() {
        super();
    }

    @Override
    public void eat() {
        System.out.println("学生可以吃");
    }

    @Override
    public void sleep() {
        System.out.println("学生可以睡觉");
    }

    @Override
    public void study() {
        System.out.println("学生可以学习");
    }
}


package wiki.mrx.www;

public class Test {
    public static void main(String[] args) {
        StudentImp s = new StudentImp();
        s.eat();
        s.sleep();
        s.study();
    }
}
