1.泛型

泛型：检测或设置引用数据类型

  • 在集合中加泛型可以规定存入集合的数据类型，便于后续遍历
  • 在类中加泛型可以后期传入具体的数据类型，而不必提前规定
package wiki.mrx.www;

class Testa<E> {
    public void test(E m){

    }
}

public class Test {
    public static void main(String[] args) {
       Testa<String> t = new Testa<>();
       t.test("test");
    }
}

4.枚举

枚举类enum：可创建的对象个数是有限的，确定的

package wiki.mrx;

public class Season {
    private final String seasonName;
    private final String seasonDesc;

    private Season(String seasonName, String seasonDesc) {
        this.seasonName = seasonName;
        this.seasonDesc = seasonDesc;
    }

    public static final Season SPRING = new Season("春天","春暖花开");
    public static final Season SUMMER = new Season("夏天","夏日炎炎");
    public static final Season AUTUMN = new Season("秋天","秋高气爽");
    public static final Season WINTER = new Season("冬天","寒冬腊月");

    public String getSeasonName() {
        return seasonName;
    }

    public String getSeasonDesc() {
        return seasonDesc;
    }

    @Override
    public String toString() {
        return "Season{" +
                "seasonName='" + seasonName + '\'' +
                ", seasonDesc='" + seasonDesc + '\'' +
                '}';
    }
}

5.反射reflect

反射：动态获取和调用类或对象信息的机制主要了解以下5类：

  • Class类
  • ClassLoader类
  • Constructor类
  • Methods类
  • Field类
反射获取字节码对象

类加载器(ClassLoader)：将.class文件加载到内存生成class对象

获取class对象方法：
1.getClass()
2.类名.class
3.Class.forName("类的正名")

Student stu = new Student();
Class c1 = stu.getClass();
Class c2 = Student.class;
Class c3 = Class.forName("com.company.Student");

反射获取构造器对象

Class c=Class.forName(正名)

c.getConstructor()公共无参构造

Constructor c2=c.getConstructor(String.class)公共有参构造

c.getDeclaredConstructor(int.class)私有有参构造

Constructor[] cs=c.getConstructors()所有构造

c2.getName()获取构造器名字

Student stu=(Student) c2.newInstance(张)创造对象

//awt例子
package wiki.mrx;

import java.awt.*;
import java.awt.event.WindowAdapter;
import java.awt.event.WindowEvent;

public class TestGUI {
    public static void main(String[] args) {
        //1.新建窗口和面板
        Frame frame = new Frame();
        Panel panel = new Panel();

        //2.窗口属性设置
        frame.setLayout(null);
        frame.setBounds(300,300,500,500);
        frame.setBackground(new Color(3,6,9));

        //3.面板属性设置
        panel.setBounds(50,50,400,400);
        panel.setBackground(new Color(0, 251, 141));

        //4.窗口中放置面板
        frame.add(panel);
        frame.setVisible(true);

        //5.创建监听事件
        frame.addWindowListener(new WindowAdapter() {
            @Override
            public void windowClosing(WindowEvent e) {
                System.exit(0);
            }
        });
    }
}
