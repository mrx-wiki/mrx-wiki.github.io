1.html

html是Tim Berners-Lee1990年创造的标记语言

HTML 1.0——1993年6月——IETF草案
HTML 2.0——1995年1 1月——RFC 1866
HTML 3.2——1997年1月——W3C标准
HTML 4.0——1997年12月——W3C标准
HTML 5.0——2008年5月——W3C标准

1️⃣标准：https://html.spec.whatwg.org/multipage/

2️⃣菜鸟：https://www.runoob.com/html/html-tutorial.html

3️⃣w3cschool：https://www.w3school.com.cn/html/index.asp

2.css

css是w3c在1996年创造的层叠样式表

CSS 1.0——1996年12月——W3C标准
CSS 2.0——1998年5月——W3C标准
CSS 3.0——2001年5月——W3C标准

1️⃣标准：https://www.w3.org/Style/CSS/

2️⃣菜鸟：https://m.runoob.com/css/css-tutorial.html

3️⃣w3cschool：https://www.w3school.com.cn/css/css_jianjie.asp

JS1.3——1998年——ES2
JS1.5——1999年——ES3
JS1.8——2009年——ES5

ES2017—2017年——ES8
ES2018—2018年——ES9
ES2019—2019年——ES10
ES2020—2020年——ES11
ES2021—2021年——ES12
